## 这个项目的说明

这个项目是我用来学习php的，从零开始。为什么想要学习PHP，当然是因为PHP是世上最好的语言...开玩笑拉。当初想学习PHP，只是因为做前端的时候，一直不怎么了解后台到底是怎样一个过程，所以很想了解后台，而PHP很适合网站开发，所以就选择了学习PHP.
